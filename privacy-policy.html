개인정보처리방침

정부광고-클립보드는 사용자의 복사된 번호(의뢰번호, 수탁번호, 수정요청) 정보를 브라우저 내부 저장소에 저장합니다.

이 정보는 로컬에서만 사용되며, 외부 서버로 전송되지 않으며 제3자에게 제공되지 않습니다.

사용자는 확장 프로그램에서 언제든지 데이터를 삭제할 수 있습니다.

storage 권한 사용 근거

사용자 설정 및 확장 프로그램의 상태를 저장하기 위해 storage 권한을 사용합니다. 사용자가 설정한 옵션, 최근 검색 기록, 또는 클립보드 히스토리 등을 로컬에서 저장하여 사용성을 개선합니다. 이 데이터는 모두 사용자의 브라우저 내에서만 관리되며, 개인 정보는 외부 서버로 전송되지 않습니다.

alarms 권한 사용 근거

사용자가 지정한 시간에 알림을 예약(schedule reminders)하기 위해 alarms API 권한을 사용합니다.

notifications 권한 사용 근거

예약된 시간에 사용자에게 알림(notification)을 표시하기 위해 notifications API 권한을 사용합니다.

downloads 권한 사용 근거

확장 프로그램에서 사용자가 기록한 데이터를 CSV 형식으로 내보내기 위해 downloads 권한을 사용합니다. 사용자가 버튼을 클릭하면 해당 데이터가 로컬 파일로 저장되며, 어떠한 개인정보도 외부 서버로 전송되거나 수집되지 않습니다.

호스트 권한 사용 근거

사용자가 보고 있는 특정 웹페이지에서 데이터를 복사하거나, 지정된 페이지의 정보를 클립보드 형태로 복사·저장하기 위해 해당 호스트에 대한 액세스 권한이 필요합니다. 예를 들어, 특정 웹사이트의 의뢰번호, 수탁번호, 세금계산서 등을 빠르게 추출하여 효율성을 높이기 위한 기능입니다. 사용자가 명시적으로 지정한 사이트에서만 동작하며, 그 외의 사이트에선 어떤 데이터도 접근하지 않습니다.
